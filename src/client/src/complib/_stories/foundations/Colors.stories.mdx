# Colors

## Overview

The type library uses several colors to help users identify statuses, actions, and navigate the application in general.
Below you will find an overview of all the colors available in the design system, and that are consumable by the developers.

The structure and form of the color palette is highly inspired by Material Design 3 which one can read more about here: https://m3.material.io/styles/color/overview
