# Shadows

## Overview

Here you will find an overview of the different shadows available. These are used to create a sense of hierarchy and elevation within the UI and different layouts.

## Shadow levels

<Box
  display="flex"
  flexDirection={"column"}
  justifyContent={"space-between"}
  gap={theme.spacing.xxl}
  p={theme.spacing.xl}
  border={`2px solid ${lightTheme.outline.base}`}
  borderRadius={theme.border.radius.medium}
>
  {Object.entries(theme.shadow).map(([k, v]) => {
    return (
      <Box key={v} display="flex" flexDirection={"column"} gap={theme.spacing.large} p={theme.spacing.large} boxShadow={v}>
        <Text variant={"title-medium"}>theme.shadow.{k}</Text>
        <Text variant={"label-large"}>box-shadow: {v}</Text>
      </Box>
    );
  })}
</Box>
