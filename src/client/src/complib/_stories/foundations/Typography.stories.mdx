## Text and heading variants
There are 15 styles which make up the type scale, they fall into 4 different categories: _Display, Headline, Title, Label and Body_.
These are meant to support a wide range of use cases and can be used in combination with multiple different html elements.
The components "text" and "heading" support a variant parameter in the format of ___variant={"(category)-(size)"}___,
which will apply the type styles according to what is shown below. When this is used in combination with the polymorphic "as" parameter;
it allows the consumer to use semantically correct html and apply the styling which best suits the given use case.
