# Spacing

## Overview

Below you will find an overview of all the spacing primitives available, each step increasing in relation to the one that came before it.
The spacing is calculated from the base font size (1rem), which means that altering the base font size will scale the spacing accordingly.

## Spacing sizes

<Box
  display="flex"
  flexDirection={"column"}
  gap={theme.spacing.multiple(8)}
  p={theme.spacing.multiple(5)}
  border={`2px solid ${lightTheme.outline.base}`}
  borderRadius={theme.border.radius.medium}
>
  {Object.entries(theme.spacing).map(([k, v]) => {

        <Text variant={"title-medium"}>{k}</Text>
        <Text variant={"label-large"}>{v}</Text>
      </Flexbox>
    );
  })}
</Box>
